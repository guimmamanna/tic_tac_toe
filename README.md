# Love Tic Tac Toe

Human hearts vs. random LLM romantics in a cozy, web-deployable GUI.

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

- Streamlit Community Cloud: push this repo and select `app.py`.
- Any Python host: run `streamlit run app.py` with port `8501`.
