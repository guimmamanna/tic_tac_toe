# Sound Files

This directory contains sound effects for the game.

## Files:
- `move.mp3` - Sound when a player makes a move
- `win.mp3` - Sound when a player wins
- `draw.mp3` - Sound when the game ends in a draw
- `click.mp3` - General click sound
- `notification.mp3` - Notification sound

## Adding Real Sounds:
Replace these placeholder files with actual .mp3 sound files for the full experience.

Recommended sources for free sounds:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/
